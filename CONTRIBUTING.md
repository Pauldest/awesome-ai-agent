# 🤝 贡献指南 (Contributing Guide)

感谢你有兴趣为 **Awesome AI Agent Learning** 做出贡献！每一份推荐都让这个资源库变得更好。

## 📋 提交规范

### 推荐新资源

请通过 Pull Request 提交，格式如下：

```markdown
* 🟢 [**资源名称** (年份): 完整标题](链接) - 一句话中文描述。 `[标签1] [标签2]`
```

**必须包含：**
- ✅ 资源的完整链接（确保可访问）
- ✅ 一句简短准确的**中文描述**
- ✅ 难度等级标注：🟢 入门 | 🟡 进阶 | 🔴 高级
- ✅ 合适的标签（如 `[Reasoning]`、`[Multi-Agent]`、`[Tool Use]` 等）

### 资源质量标准

我们欢迎以下类型的资源：
- 📄 与 AI Agent 强相关的**学术论文**
- 📝 高质量的**技术博客和深度解读**
- 🛠️ 实用的**框架、工具和开源项目**
- 📺 优质的**视频课程和教程**
- 📖 相关的**书籍推荐**

**请确保：**
- 资源与 AI Agent 主题强相关
- 资源质量高，有实际学习价值
- 链接有效且可访问
- 不重复已有的资源

## 🏷️ 标签使用规范

常用标签列表：

| 标签 | 适用场景 |
| :--- | :--- |
| `[Reasoning]` | 推理相关 |
| `[Planning]` | 规划相关 |
| `[Tool Use]` | 工具调用相关 |
| `[Multi-Agent]` | 多智能体相关 |
| `[Memory]` | 记忆机制相关 |
| `[Reflexion]` | 反思/自我改进相关 |
| `[Architecture]` | Agent 架构设计 |
| `[Simulation]` | 仿真/模拟环境 |
| `[Embodied]` | 具身智能 |
| `[Evaluation]` | 评估/基准测试 |

## 🐛 报告问题

如果你发现：
- 🔗 失效的链接
- ❌ 不准确的描述
- 📂 资源分类不当

请[创建一个 Issue](../../issues/new) 告诉我们。

## 💬 讨论与建议

对于项目结构、新板块等建议，欢迎在 [Discussions](../../discussions) 中提出。
